Tutorial do formulário (este tutorial é dinâmico e sempre serão implementadas novas orientações)

1 - Para utilizar este template serão necessários os plugins mencionados na página principal. Estes plugins serão atualizados conforme necessidade e será informada neste tutorial a atualização do mesmo.

2 - Após configurar os plugins corretamente no código de desenvolvimento, você pode inserir todo o conteúdo na tag 'body' do seu código e já poderá visualizar o template no seu navegador.

3 - Caso queira inserir uma nova etapa, você deve replicar um elemento 'li' dentro da 'ul' na linha 156, assim como os existentes na lista e dar a ele seu respecitvo ID.

4 - Caso queira inserir elementos de formulário dentro da nova etapa, você deve inserir na div de classe 'tab-content', na linha 193, uma div com a classe 'tab-pane' e o mesmo ID que deu na 'li' criada.
